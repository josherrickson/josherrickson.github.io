Getting WICD to connect to MWireless

Note: There's apparently a bug which results in "Bad Password" errors. Using PEAP with GTC may work.
  1. Save the file peap-mschapv2 in /etc/wicd/encrytion/templates.
  2. In the same directory, append the line "peap-mschapv2" to the end of file active.
  3. Restart WICD, and when connecting to MWireless, choose the new PEAP-MSCHAPV2 encryption option. Identity and Password are your Kerberos account, and Path to CA cert is the path to the security certificate available here.

Reference: http://kb.iu.edu/data/bbtc.html
